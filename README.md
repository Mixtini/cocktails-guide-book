# cocktails-guide-book
A cocktails guide book create by OverPartyLab.

Demo page: [Cocktails Guide Book](https://overpartylab.github.io/cocktails-guide-book/)


## Web app

Source code in /app.

* Install package

```sh
yarn install
```
or
```sh
npm install
```

* Run the react app
```sh
yarn start
```
or
```sh
npm run start
```
